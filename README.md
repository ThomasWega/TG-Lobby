# TGLobby
Lobby for TrustGames.net

###### Configs:
- Join/Leave messages
- Double Jump
- PiggyBack
- Build Command messages
- Spawn Command messages

###### Features:
- Hotbar Items
- Join/Leave messages send
- Double Jump
- PiggyBack
- Build/Place protection
- Spawn teleport
- XP Bar show level + update

###### Commands:
- /build <name>
- /setspawn
- /spawn <name>
